# How To Run Program

Pada Terminal gunakan gcc main.c fungsi.c -o program_tugas kemudian ./program_tugas. atau gunakan mingw32-make gcc -Wall main.c fungsi.c -o uasTim9 kemudian ./uasTim9.

ketika program dijalankan akan menampilkan menu utama yang memiliki beberapa fungsi. berikut adalah fungsi fungsi setiap Pilihan:
1. pada pilihan pertama kita bisa menginput nama tugas, deadline, bobot nilai, dan lama pengerjaan tugas. fungsi ini sangat penting untuk dapat menjalankan fungsi pada pilihan lain
2. pada pilihan kedua, kita dapat melihat semua tugas yang sudah kita input yang diurutkan berdasarkan deadline
3. pada pilihan ketiga, kita dapat mencari tugas berdasarkan keyword
4. pada pilihan keempat, kita dapat melihat rekomendasi tugas berdasarkan skor prioritas
5. pada pilihan kelima, terdapat sub menu yang pada pada sub menu tersebut kita dapat menandai tugas selesai atau mengundo tugas selesai tersebut
6. pada pilihan keenam, program akan keluar dan membersikan semua memori yang digunakan program
